Linux ps命令

发表于 2018-01-04 | 分类于 Linux | | 阅读次数:

Linux ps命令用于显示当前进程(process)的状态

语法

ps [options] [--help]

参数

ps 的参数非常多, 在此仅列出几个常用的参数并大略介绍含义

  • -A 列出所有的行程
  • -u 显示指定用户的进程
  • -e 等于“-A”
  • -f 以格式化显示
  • a 显示所有涉及终端的进程信息
  • u 显示面向用户的输出
  • x 显示没有终端的进程

au(x) 输出格式 :

USER PID %CPU %MEM VSZ RSS TTY STAT START TIME COMMAND
